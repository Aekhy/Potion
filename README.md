# Potion

Le but du jeu est de créer des potions ! Vous devrez essayer de comprendre comment les éléments du jeu interagissent les uns avec les autres pour trouver ces précieux élixirs.

Important : Lisez "Avant de commencer" pour éviter de perdre votre progression.
## Touches
### Clavier :

    échappe / o - ouvrir/fermer le menu d'accueil
    tab - ouvrir/fermer le menu de gestion
    i - ouvrir/fermer l'inventaire (n'est pas souvent disponible, voire pas du tout)

### Souris :

    Cliquer :
        left click/LMB - Activer l'effet d'un bouton, sélectionner un onglet.
    Maintenir :
        left click/LMB - Drag / Tenir UN item
        right click/RMB - Drag / Tenir le "stack" d'item
    Relâcher :
        Si vous tenez un item, relâcher le bouton pose l'item au niveau de votre curseur. Si l'item ne peut pas être posé à cet endroit, il revient à sa position d'origine.

## Avant de commencer.

    Le système de sauvegarde de ce jeu sauvegarde uniquement les points suivants:

        1. Les items dans votre inventaire
        2. L'état de vos connaissances

    Remarque :
    Si vous déposez un item dans le chaudron, un outil, ou tout slot autre qu'un slot de votre inventaire et que vous quittez le jeu, cet item sera perdu.

    Remarque :
    Si le jeu crash, considérez votre progression depuis votre dernière sauvegarde (c'est-à-dire la dernière fois que vous avez fermé le jeu sans crash) comme perdue.

Lancez le programme. Vous arrivez sur la page d'accueil.

## Accueil - "échappe / o"

    Cliquez sur "JOUER" pour lancer/continuer votre partie. 
    Cliquez sur "QUITTER" pour sauvegarder et quitter le jeu. 
    
    Affiché sur certains menus : "RETOUR AU MENU PRINCIPAL". 
    Cliquez pour revenir à ce menu. 
    
    Vous pouvez utiliser les flèches directionnelles pour choisir l'option et appuyer sur la touche "ENTREE" pour valider votre choix. 

Si cliquer sur "JOUER" ne fait rien, réessayez encore une fois.

## Page Chaudron

    La partie 'gauche' laisse place à votre inventaire. 
    La partie du 'milieu' (carré rouge) est votre chaudron.
    La partie de 'droite' permet de manipuler les recettes. 
    La partie 'en bas à gauche' permet d'aller sur le menu outils en cliquant sur "OUTILS".

## Inventaire

Votre inventaire est divisé en quatre onglets. 
Chaque onglet contient un certain nombre de pages (d'autres onglets numérotés). 
Chaque page contient une grille de slot pour stocker vos items. 
Un slot spécifique est dédié à la destruction d'item. Pour l'utiliser, déposez l'item à détruire dans le slot et cliquez sur "DETRUIRE L'ITEM".

Remarque : Pour déposer un item dans votre inventaire, vérifiez que l'onglet sélectionné (en gris foncé) correspond au type de votre item. Sinon, l'item retournera à son emplacement d'origine.

### Ingredients
    Contient vos ingrédients.
### Mixtures
    Contient vos mixtures élémentaires et astrales.
### Potions
    Contient vos mixtures étranges, potions ratées et finies.
### Recettes
    Contient votre papier, vos recettes.

## Chaudron

    Le chaudron a pour but de mélanger vos ingrédients et/ou vos mixtures astrales/élémentaires afin de créer des mixtures étranges ou des potions finies. 
    Pour ajouter des choses dans le chaudron, il suffit de faire un glisser-déposer de l'item au-dessus du carré rouge. 
    Les items que vous pouvez mettre dans le chaudron dépendent de ce qui est déjà présent dans le chaudron et de l'item en question.
    Pour récupérer le contenu du chaudron, cliquez sur "FINIR". Le résultat apparaîtra dans la case au-dessus. 
    Vous pourrez alors récupérer le contenu avec un glisser-déposer dans votre inventaire.

## Recettes

    Vous pouvez voir une recette dans cette partie. 

    Pour voir le contenu d'une recette sans modification, faites un glisser-déposer d'une recette dans la case grise en dessous de "VISUALISER". 
    Ceci affichera toujours le contenu de ce qu'il y a dans cette case. 

    Pour écrire / réécrire une recette, faites un glisser-déposer du papier ou d'une recette déjà écrite dans la case en dessous de "ECRIRE". La recette en cours d'écriture, c'est-à-dire le contenu actuel de votre chaudron, sera affichée jusqu'à ce que vous récupériez le contenu du chaudron. Dans ce cas, le contenu précédent restera affiché.

    Vous pouvez placer à tout moment du papier ou une recette dans les deux cases. 

    Important : Comprenez que tout ce qui est dans la case en dessous de "ECRIRE" effacera les données de l'item et repartira avec le contenu du chaudron. 

    Remarque : Si vous avez des items dans les deux cases, c'est l'item dans "VISUALISER" qui sera affiché.

## Page Outil

    La partie 'gauche' laisse place à votre inventaire. 
    La partie 'droite' répertorie les outils.

### Utiliser un outil

    Vous pouvez faire un glisser-déposer d'une mixture (astrale, élémentaire ou étrange) sur un des carrés de couleur représentant un outil. Si l'item revient à sa position d'origine, c'est que l'outil ne peut être utilisé sur celui-ci (c'est le cas si l'on met une mixture astrale dans un outil ne fonctionnant que sur des mixtures élémentaires, ou inversement). 
    Une fois l'item dans l'outil, vous pouvez cliquer sur le texte décrivant l'effet (détaillé plus bas) pour l'appliquer à votre item. Celui-ci s'affiche alors dans la case grise correspondant à l'outil.


## Ce qu'est une potion
### États d'une potion

    Dans ce jeu, une potion a trois états possibles :

        Une potion finie, elle porte un nom unique et identifiable.
        Une potion ratée, décrite comme telle.
        Une mixture étrange, c'est presque une potion, mais il manque un petit quelque chose...

### Composition d'une potion

    Ainsi, une potion finie ou ratée est composée de trois éléments :

        Une mixture astrale
        Une mixture élémentaire
        Une propriété alchimique

    La combinaison de ces trois facteurs détermine le résultat final.

### Mixture astrale/élémentaire

    Une mixture (astrale ou élémentaire) est un mélange d'ingrédients du même type. On l'obtient en mettant des ingrédients dans le chaudron. Dans le jeu, vous devrez découvrir quels ingrédients font partie de quelle catégorie afin de les mélanger ensemble pour créer la mixture souhaitée.

### Mixture astrale

    Certains ingrédients mélangés ensemble donneront une mixture astrale. Par défaut, ce sera la mixture astrale "ORIGINE". Avec une combinaison d'ingrédients appropriée, vous obtiendrez une mixture astrale du nom d'une planète.

### Mixture élémentaire

    Certains ingrédients mélangés ensemble donneront une mixture élémentaire. Par défaut, ce sera la mixture astrale "NEUTRE". Avec une combinaison d'ingrédients appropriée, vous obtiendrez une mixture élémentaire du nom d'un élément.

### Effet

    Un effet est l'action d'un outil sur une mixture (astrale, élémentaire ou étrange).
    Un effet est applicable soit sur une mixture astrale, soit sur une mixture élémentaire (pas les deux). 
    Un seul effet peut être appliqué par mixture. 
    Un effet ne peut pas être retiré.

### Propriété alchimique

    Une propriété alchimique est une combinaison d'un effet d'une mixture astrale et d'un effet d'une mixture élémentaire. Toutes les combinaisons correspondent à une propriété alchimique, mais ce ne sera peut-être pas celle qui permettra d'obtenir une potion finie.

## Faire une potion
### Première méthode

    Une première approche est de créer vos mixtures astrales et élémentaires séparément, de leur appliquer un effet, puis de les mélanger dans votre chaudron pour obtenir une potion finie ou ratée.

### Seconde méthode

    La seconde méthode consiste à mélanger tous vos ingrédients en une seule fois dans le chaudron, vous obtiendrez alors une mixture étrange. 
    Il reste ensuite à appliquer une propriété alchimique sur celle-ci : pour cela, il faut appliquer une combinaison de deux effets à votre mixture étrange. Les effets s'appliqueront directement à la mixture (astrale ou élémentaire) correspondante. Vous aurez alors une potion finie, ou ratée.

### Quelle méthode ?

    La première est plus "longue" mais permet d'avoir une meilleure compréhension des potions et d'obtenir la recette complète. La seconde est plus "rapide" mais requiert une meilleure connaissance et ne permet pas d'obtenir la recette complète (en effet, on ne peut pas encore ajouter une recette au outils, on obtient donc une recette complète en combinant une mixture astrale avec effet et une mixture élémentaire avec effet dans le chaudron pendant qu'un papier/recette est en train d'être écrit.)

## Obtenir la bonne mixture astrale/élémentaire

    Chaque ingrédient possède des caractéristiques. En ajoutant un ingrédient, vous ajoutez ces caractéristiques à l'astre/l'élément représenté dans votre chaudron. Cela vous emmène sur un autre astre/élément ou ne change rien. 
    Une connaissance de chaque ingrédient permet de savoir quelle caractéristique interagit avec quelle caractéristique. Vous pourrez alors remplacer une combinaison d'ingrédients par une autre combinaison à caractéristique équivalente, ce qui permet de créer une même potion finie avec plusieurs recettes différentes.
